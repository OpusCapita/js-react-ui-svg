# OpusCapita react-svg

## Synopsis

React components for easy work with [**Scalable Vector Graphics**](https://en.wikipedia.org/wiki/Scalable_Vector_Graphics)
For components overview:
* Clone repository
* Run `npm install`
* Run `npm start`
